# Todo

### Requirements
- iOS 10.3+
- Xcode 9.4+
- Swift 4+

### Installation 
- install cocoa pods
`gem install cocoapods`
- then update your pod repository
`pod update`
- install your project pods 
`pod install`
- open - Todo.xcworkspace file
- clean the project **'Command + Shift + K'**
- **run** your projects
